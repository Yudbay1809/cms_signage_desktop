# Contributing

Thanks for contributing to CMS Signage Desktop.

## Setup
1. Install Flutter stable (`3.38.x`).
2. Run `flutter pub get`.
3. Ensure backend API is reachable for manual testing.

## Before Opening PR
- `flutter analyze`
- `flutter test`

## PR Checklist
- [ ] Focused scope
- [ ] No unrelated refactor
- [ ] Docs updated if behavior changed
- [ ] UI screenshots added if relevant
